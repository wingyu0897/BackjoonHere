# [문제](https://www.acmicpc.net/problem/2675 "#2675번")
  
> 알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.
<hr/>

# 풀이

```cpp
#include <iostream>
using namespace std;

int main() 
{
    int count;
    cin >> count;

    for (int i = 0; i < count; i++)
    {
        int times;
        string input;
        cin >> times >> input;
        for (int j = 0; j < input.length(); j++)
        {
            for (int k = 0; k < times; k++)
            {
                cout << input.at(j);
            }
        }

        cout << '\n';
    }

    return 0;
}
```
