# [문제](https://www.acmicpc.net/problem/9086 "#9086")
  
> 문자열을 입력으로 주면 문자열의 첫 글자와 마지막 글자를 출력하는 프로그램을 작성하시오.
<hr/>

# 풀이

```cpp
#include <iostream>
using namespace std;

int main() 
{
    int count;
    cin >> count;
    for (int i = 0; i < count; i++)
    {
        string str;
        cin >> str;
        cout << str.at(0) << str.at(str.length() - 1) << endl;
    }
    return 0;
}
```
